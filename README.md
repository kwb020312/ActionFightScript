# 💥JavaScript를 활용한 액션 대전게임을 만들어보자!

[javascript를 활용한 대전 액션게임 하러가기!](https://action-fight-script.vercel.app/)

---

### 🎮조작법

- Player1
  1. 이동
    - `w`, `a`, `d`
  2. 공격
    - `space bar`
- Player2
  1. 이동
    - `↑`, `←`, `→`
  2. 공격
    - `↓`
